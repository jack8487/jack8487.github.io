yolo算法初学

yolo算法

指标分析

$$
Precision = {TP\over{TP + FP}}
$$

$$
Recall={TP\over{TP+FN}}
$$

$$
Accuracy = \frac{TP+TN}{TP+FP+FN+TN}
$$

$$
F1_{score} = \frac{2PrecisionRecall}{Precison+Recall}
$$

相关(Relevant),正类 无关(NonRelevant),负类
被检索到(Retireved) true positives(TP) false positives(FP)
未被检索到(Not Retireved) false positives(TP) true negatives(TN)

T/N：表示预测的对错

P/N：表示预测的结果

$$
IoU = \frac{Area \ of \ Overlap}{Area \ of \ Union}
$$

  • if IoU >= 0.5 True Positive(TP)
  • if IoU < 0.5 False Positive(FP)
  • 如果存在却没有检测到 False Negative(FN)
  • 检测到了但是没有用处 True Negative(TN)
  • AP衡量的是学习出来的模型在每个类别上的好坏(Average Precision)
  • MAP衡量的是学出的模型在所有类别上的好坏。
